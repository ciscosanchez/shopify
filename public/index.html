💡 Help & Information

Getting Shopify API Credentials

  1. Go to Shopify Admin
  2. Navigate to Settings → Apps and integrations
  3. Click Develop apps
  4. Create a new app named "Product Importer"
  5. Go to Configuration → Admin API
  6. Enable: write_products and read_products
  7. Copy your Access Token

CSV Format

Your CSV should have a header row with these columns:

Column Required? Description
title ✓ Product name
description Product description
vendor Brand/manufacturer
sku Stock keeping unit
price Selling price
compare at price Original price (shows discount)
inventory quantity Starting stock
image url Product image URL

JSON Format

JSON should be an array of product objects:

[
  {
    "title": "Product Name",
    "price": "99.99",
    "sku": "SKU-001"
  }
]

🔒 Your credentials are only used to connect to your Shopify store and never stored.
